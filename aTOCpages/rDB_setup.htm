Extraction Setup

Administrator instrumentation in is built right into the interface. Simple settings in user accounts determine whether a user has access to all or a subset of administrative instrumentation, or only to daily business use such as table view, spreadsheet and pivot tables and charts using the extracted report data. This guide gives instructions on how to set up extraction templates using reports, where to create scripts for customized automated extractions from to the output you need, and a bit about using the built-in spreadsheet and pivot table and chart features.
